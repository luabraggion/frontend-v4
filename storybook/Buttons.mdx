## 1. Button

Botão principal, com variantes e tamanhos.

### Exemplo básico

# Button

Buttons permitem que usuários disparem uma ação ou evento com um único clique. Por exemplo, você pode usar um botão para submeter um formulário, abrir um diálogo, cancelar uma ação ou executar uma operação importante.

<Button variant="warning">Warning</Button>
```

## Usage

- Botões podem conter ícone à esquerda ou direita
- Use espaçamento de 8px entre botões
- Substitua o texto por um loader se a ação estiver processando
- A largura do botão é definida pelo conteúdo, evite definir largura fixa
- Use apenas um botão primário por tela, os demais devem ter menor destaque

> 💡 **Dica:** Para botões apenas com ícone, utilize o componente `ButtonIcon`.

---

## Accessibility

- Use `id` para garantir associação correta de rótulo
- Sempre forneça texto descritivo ou use `ariaLabel` para botões só com ícone
- Use `ariaLabel` para nome acessível customizado
- Use `ariaLabeledBy`, `ariaHasPopup`, `ariaExpanded`, `ariaControls`, `aria-describedby`, `aria-pressed` conforme necessário

## 3. ButtonWithIcon

Botão com ícone e texto, com ícone à esquerda ou direita.

### Exemplo

```tsx
<ButtonWithIcon icon={<EditIcon />}>Editar</ButtonWithIcon>
<ButtonWithIcon icon={<TrashIcon />} iconPosition="right">Excluir</ButtonWithIcon>
```

### Propriedades

| Propriedade   | Tipo              | Descrição                     |
| ------------- | ----------------- | ----------------------------- |
| icon          | ReactNode         | Ícone exibido                 |
| children      | ReactNode         | Texto do botão                |
| iconPosition  | 'left' \| 'right' | Posição do ícone              |
| iconClassName | string            | Classe CSS extra para o ícone |
| ...           | Button props      | Todas as props do Button      |
